---
title: Wallet Management 
---
The piggyvest Wallet API is designed to empower businesses and users to efficiently create, manage, and interact with their wallets. 
This API equips users with tools for handling funds, accruing interest, conducting transactions, etc within the Piggyvest Business ecosystem.

## Key Features

- Interest Accrual: Wallets can accrue interest, providing an opportunity for users to grow their funds. Businesses have the flexibility to set the interest rates for their customers' wallets.
- Businesses can carry out the following using the wallets APIs
    - Fetch a Balance
    - Top up their wallet balance
    - Increase/Decrease Wallet Balance of their customers
    - Payouts
