---
title: Retrive wallet account numbers
---

To retrieve all virtual account numbers that belong to a wallet make an api request to the retrieve wallet account numbers endpoint.

Path Parameter
| Parameter  | Description | Mandatory |
| --- | --- | --- |
| wallet_id| The unique ID of the wallet | Required |

The request should be like
```tsx Request
import axios from 'axios'

const secretKey = process.env.PVB_SECRET_KEY
const url = ${process.env.PVB_BASE_URL}/api/v1/wallet/:wallet_id/accounts

await axios.get(url, {
    headers: {
      'Authorization': `Bearer ${secretKey}`,
    },
  })
```
