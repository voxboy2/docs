---
title: Retrive a wallet
---

To retrieve a wallet you need to make an api call with it's unique ID.

Path Parameter
| Parameter  | Description | Mandatory |
| --- | --- | --- |
| wallet_id| The unique ID of the wallet | Required |

The request should be like

```tsx Ts
import axios from 'axios'

const secretKey = process.env.PVB_SECRET_KEY
const url = ${process.env.PVB_BASE_URL}/api/v1/wallet/:wallet_id

await axios.get(url, {
    headers: {
      'Authorization': `Bearer ${secretKey}`,
    },
  })
```
